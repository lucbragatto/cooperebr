## Características do regime — A CONFIRMAR

A definição precisa depende do tipo de consórcio do Sinergia Ambiental:

### Hipótese A — Consórcio Empresarial (Lei 6.404/76 Arts. 278-279)
- Mais provável para o caso
- Não tem personalidade jurídica própria (Art. 278 § 1º)
- Sociedades consorciadas mantêm autonomia
- Contrato regula direitos e obrigações entre consorciados
- Regime tributário próprio (não é cooperativa)

### Hipótese B — Consórcio de Bens (Lei 11.795/2008)
- Aplicável a aquisição de bens via consórcio (carros, imóveis)
- Pouco provável para energia
- Tem regulação rigorosa do Banco Central

### Hipótese C — Outro tipo
- Consórcio público (Lei 11.107/2005)
- Consórcio intermunicipal
- Outras configurações

**Pendente:** Luciano confirmar qual regime aplicável ao Sinergia Ambiental.

## Distinções importantes vs CoopereBR

| Característica | CoopereBR (cooperativa) | Sinergia (consórcio assumido) |
|---|---|---|
| Lei | 5.764/71 | 6.404/76 Art. 278-279 (assumido) |
| Tipo de relação | Ato cooperativo (não-mercantil) | Contrato consorcial (civil/comercial) |
| Personalidade jurídica | Sim (cooperativa singular) | Não (em consórcio empresarial puro) |
| Direito a sobras | Sim | Não (depende do contrato) |
| Adesão voluntária | Princípio cooperativista | Por contrato |
| Tributação | Tema 536 STF + isenção | Conforme regime |
| Foro | Vitória/ES | Vitória/ES (mesmo grupo SISGDSOLAR) |

## SISGDSOLAR neste modelo

**Citada como infraestrutura técnica** que opera a plataforma multi-tenant do Sinergia. **NÃO é parte signatária** deste termo.

A relação Sinergia ↔ SISGDSOLAR é tratada no contrato SaaS separado (`00-contratos-saas-instituicoes/`).
