## Princípios fundamentais

### 1. Bilateralidade dos termos de adesão

Cada termo de adesão é **bilateral** entre instituição parceira e membro:
- CoopereBR ↔ Cooperado (ato cooperativo Lei 5.764/71)
- Sinergia ↔ Consorciado (contrato de consórcio)
- Futuros parceiros ↔ membros (regime aplicável)

**SISGDSOLAR é citada como infraestrutura técnica**, mas **NÃO é parte signatária** desses termos.

### 2. Regimes jurídicos distintos

| Característica | CoopereBR | Sinergia |
|---|---|---|
| Tipo | Cooperativa singular sem fins lucrativos | Consórcio |
| Lei | Lei 5.764/71 | Lei 6.404/76 Arts. 278-279 ou Lei 11.795/2008 |
| Natureza | Ato cooperativo (não-mercantil) | Contrato civil/comercial |
| Direitos do membro | Voto, sobras, adesão livre | Conforme contrato de consórcio |
| Tributação | Tema 536 STF + isenção | Conforme regime |

### 3. Portabilidade entre parceiros (Camada 3)

Membros podem **migrar** entre instituições parceiras do SISGDSOLAR, observadas:
- Acordo Inter-Parceiros prévio (entre instituições)
- Termo de Transferência (assinado pelo membro)
- Carência mínima (6-12 meses na instituição atual)
- Aviso prévio (90 dias)
- Ciência expressa sobre mudança de regime jurídico
- Tratamento de saldos SCEE, créditos acumulados, etc.
