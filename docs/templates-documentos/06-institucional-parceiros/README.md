## Princípios

1. **Cada parceiro mantém seus documentos institucionais** — Estatuto da CoopereBR não vincula Sinergia, e vice-versa
2. **Documentos institucionais são propriedade da instituição** — SISGDSOLAR apenas armazena
3. **Versionamento rigoroso** — Estatuto reformado em assembleia gera nova versão
4. **Reflexo no sistema SISGDSOLAR** — alterações estatutárias podem implicar ajustes operacionais (ex: regras de aprovação, alçadas, categorias de cooperados)
