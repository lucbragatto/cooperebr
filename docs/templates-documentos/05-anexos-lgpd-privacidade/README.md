## Princípios diretivos LGPD CoopereBR

1. **Minimização** — coletar só o que é necessário
2. **Finalidade específica** — não usar dados para fins não declarados
3. **Transparência** — explicar tratamento em linguagem clara
4. **Segurança** — medidas técnicas e administrativas adequadas
5. **Direitos do titular** — facilitar acesso, correção, exclusão
6. **Compartilhamento controlado** — só com parceiros que firmaram NDA + DPA
7. **Retenção limitada** — definir prazos claros pós-rescisão

## Estrutura recomendada de governança LGPD

- **DPO (Encarregado):** designar pessoa específica
- **Comitê de Privacidade:** Presidente Conselho + 1 advogado + 1 TI
- **DPA (Data Processing Agreement)** com SISGDSOLAR (operador)
- **Treinamento anual** para equipe que lida com dados
- **Plano de resposta a incidentes** de dados
- **Auditoria anual** de conformidade
