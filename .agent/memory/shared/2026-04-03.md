### Planos e Base de Cálculo
- Base padrão SISGD: **kWh cheio** (TUSD+TE+PIS/COFINS+CIP com todos os tributos)

- Mostrar discriminado: true por padrão
- **Implementado**: schema Prisma + motor proposta + telas admin (plaid-coral)

### CooperToken — modelo definitivo
- **Valor do token**: admin escolhe fixo OU kWh apurado no mês (após pagamento da fatura)
- **Emissão**: valor gravado no ledger no momento da emissão (`valorEmissao`)
- **Desvalorização progressiva**:
  - Dia 1-10:  100% do valorEmissao
  - Dia 11-20: 90%
  - Dia 21-26: 75%
  - Dia 27-29: 50%
  - Dia 30+:   expira → split expiração
- **Split de emissão (2%)**: SISGD 1% + Fundo Clube 1%
- **Split de circulação (1%)**: SISGD 0,4% + Cooperativa 0,4% + Fundo Clube 0,2%
- **Split de expiração (100%)**: Cooperativa 40% + SISGD 30% + Fundo Clube 30%
- **Opção A**: cooperado recebe desconto direto na assinatura
- **Opção B**: cooperado paga cheio e recebe tokens (valorEmissao = kWh apurado)
- **QR Code payment**: JWT 5min, cooperado gera → parceiro lê → sistema transfere
